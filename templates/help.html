# Keys

Press Enter to refresh the chat and check for new messages.

Press F11 for the best experience.

Press Esc to go back.

# Commands

!info Displays room statistics.

!clear Deletes all messages in the current chat room.

!delete hello Deletes all messages with the word "hello" in them.

!delete 05/15/2022 Deletes all messages in the room written on that date.

!rooms List all rooms.

!room gaming Switch your room to "gaming".
